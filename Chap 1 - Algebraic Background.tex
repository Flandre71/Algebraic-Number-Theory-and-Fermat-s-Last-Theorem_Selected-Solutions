    Yes.

    In fact, for any field $F$ and $f(x) \in F[x]$. We'll prove $\exists K/F$, s.t.\ $f$ have multiple roots in $K \Longleftrightarrow f, f'$ are not relatively prime: 
    Let $\alpha$ be a root of $f$, $f(x) = (x - \alpha)g(x)$. Now $f'(x) = g(x) + (x - \alpha)g'(x)$. Substituting $x = \alpha$, we'll see that $f'(\alpha) = 0 \Longleftrightarrow g(\alpha) = 0 \Longleftrightarrow f(x)$ has multiple roots.
\end{exercise}

\begin{exercise}

    \begin{enumerate}[label=\roman*.]
        \item We can prove $\Delta$ is antisymmetric by comparing the definition of even permutations and components of $\Delta$.

        \item First, we'll prove that for $f(x_1, x_2, \ldots, x_n) \in \mathbb{Z}[x_1, x_2, \ldots, x_n]$ which is antisymmetric, $\Delta \mid f(x_1, x_2, \ldots, x_n)$: To prove this, we only have to prove that $\exists i, j \in [1, n] \cap \mathbb{Z}$, let's say $i = 1, j = 2$, $(x_i - x_j) \mid f(x_1, x_2, \ldots, x_n)$. Notice $f(x_1, x_2, \ldots, x_n) \in (\mathbb{Z}[x_3, x_4, \ldots, x_n])[x_1, x_2]$, we can have
            \[ 
                f(x_1, x_2, \ldots, x_n) = \sum_{i, j \in [0, n] \cap \mathbb{Z}} a_{ij} x_1^i x_2^j 
            \]
            where $a_{ij} \in \mathbb{Z}[x_3, x_4, \ldots, x_n]$. And because $f(x_1, x_2, \ldots, x_n) = -f(x_2, x_1, \ldots, x_n)$, $a_{ij} = -a_{ji}$, indicating $(x_1 - x_2) \mid f(x_1, x_2, \ldots, x_n)$. Thus, $\Delta \mid f(x_1, x_2, \ldots, x_n)$.

            Now, following from the hint, we consider $f / \Delta$. It is a symmetric polynomial, so it was consist of elementary symmetric polynomials.
    \end{enumerate}
\end{exercise}
